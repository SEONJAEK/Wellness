[5조 스포르타_WELLNESS]

1. 팀원 5명 / 역할

경선재(팀장)
ERD 및 DB구축, 세션, 예약현황 확인, 분석그래프 시각화, 관리자부여, 회원정보 목록 확인 기능, Q&A게시판 SmartEditor연동
김지민
로그인, 회원가입, 회원정보 수정, 아이디 찾기, 비밀번호 찾기
김효신
설계, ERD, UI화면 구성, 요구사항작성, 아이디찾기, GX프로그램 예약, 마이페이지 구현
이초롱
화면기능서 작성, 메인페이지 구현, 지점소개페이지 구현, 시설소개페이지 구현, 전체페이지 css 보완, ppt 보완
임성수
Q&A 게시판 구현(CRUD), SmartEditior연동

2. 제목: 시간적 여유가 없는 직장인들을 위한 맞춤형 예약프로그램 사이트 "WELLNESS"

3. 개요: 
관리자 입장에서는 "그래프 시각화를 통한 회원관리"를 제공하고 
방문예약이 어려운 회원의 입장에서는 
"GX프로그램을 온라인으로 예약하여 들을 수 있는 기회를 제공" 하는 것입니다. 

4. 구현기능:
회원 - 로그인, 회원가입, 회원정보 수정, 아이디 찾기(이메일 인증), 비밀번호 찾기 기능
게시판 - Q&A 게시판 글 쓰기, 글 수정, 글 읽기, 글 삭제, 글 목록, 댓글달기, 스마트에디터, 페이징 기능
예약 - G.X예약, 예약 인원수 확인 및 예약 중복확인, 예약 확인 페이징 기능
회원관리 - 예약현황 확인, 분석그래프 시각화, 관리자부여, 회원정보 목록 확인 기능

5. 설계의 주안점: 사용자 편의에 맞는 예약 및 예약 취소 기능 구현, API를 활용한 데이터 시각화

6. 사용기술 및 개발환경

서버: apache-tomcat-9.0.41
JAVA EE IDE: Eclipse (2019-09 release)
Database: Oracle SQL Developer(version 20.2), Oracle Database (11g express edition)
사용 언어: Java( version 10.0.1), HTML, CSS, JavaScript, JSP, SQL
클라이언트 프레임워크: JQuery 3.3.0, Bootstrap 3.3.2
백앤드 프레임워크: Spring Framework
버전관리: Git
